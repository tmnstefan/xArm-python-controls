# Utilities

This page contains information on all functions present within [util.py](https://github.com/tmnstefan/xArm-python-controls/blob/main/actions/util.py) and aims to give a clear understanding both of what these functions do and the requirements for any data to be input into them.

Most of these functions contain actions for the arm that do not necessarily fit into other categories such as connection and checking verification.
