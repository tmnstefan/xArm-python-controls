# Settings

This page contains information on all functions present within [settings.py](https://github.com/tmnstefan/xArm-python-controls/blob/main/actions/settings.py) and aims to give a clear understanding both of what these functions do and the requirements for any data to be input into them.

These functions will allow you to get and change the value of a variety of settings for the arm.
